
# Space Missions Analysis Dashboard (Tableau)

This project is a Tableau dashboard created to explore and visualize space mission data.  
The dashboard provides insights into launch sites, mission outcomes, environmental conditions, and payload statistics.

---

## Tools Used
- Tableau Public
- CSV Dataset

---

## Dashboard Overview

The dashboard analyzes different aspects of space missions and launch conditions. It helps in understanding how environmental factors and launch sites relate to mission outcomes.

The dashboard includes the following visualizations:

- Average Temperature by Launch Site
- Average Humidity by Launch Site
- Wind Speed Distribution
- Mission Status by Company
- Launch Vehicle Information
- Payload to Orbit Analysis
- Mission Track Records

---

## Key Insights

- Launch conditions such as temperature, humidity, and wind speed vary across different launch sites.
- Mission outcomes differ between companies and launch locations.
- Payload mass and launch vehicles provide insights into mission capabilities.
