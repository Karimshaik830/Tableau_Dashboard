# COVID-19 Dashboard (Tableau)

This project is a Tableau dashboard that visualizes COVID-19 data to understand global trends in cases, deaths, and recoveries.

## Tools Used
- Tableau Public
- CSV dataset

## Dashboard Features
- COVID-19 cases by country
- Deaths and recoveries analysis
- Trend of cases over time
- Regional comparisons
